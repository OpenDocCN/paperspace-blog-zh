Gradient experiments 中的代码查看器现在包括一个 visual diff 工具，面向开发人员，用于更好地可视化实验中的代码更改。

## 它是如何工作的

diff 视图帮助您比较 Gradient 中的文件、目录和版本控制项目。它提供了两个文件的双向比较，使理解代码更改更加容易。
